# Dockerized Jenkins Server

## Project Overview

This project demonstrates how to run Jenkins inside a Docker container and automate the build process of a Spring Boot application.

## Technologies

- Docker
- Jenkins
- Maven
- Java
- Git
- GitHub
- Spring Boot

## Features

- Jenkins running inside Docker
- Automated GitHub cloning
- Maven build automation
- Spring Boot JAR generation
- Docker-based service management

## Build Pipeline

GitHub → Jenkins → Maven → Spring Boot JAR
